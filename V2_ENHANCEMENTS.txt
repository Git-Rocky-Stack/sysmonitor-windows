v2.2.0 - Advanced Game Mode, FPS Overlay & Fahrenheit Temps
================================================================================

NEW FEATURES
------------

* Auto Game Mode - Automatically enables Game Mode when games are detected
  - 50+ predefined popular games (Valorant, CS2, GTA5, Fortnite, Apex Legends, etc.)
  - Custom game support - add your own games
  - Background process monitoring with 2-second scan interval

* Performance Profiles - Save and switch between optimization presets
  - Maximum Performance (kills all background apps)
  - Balanced Gaming (keeps essential apps)
  - Streaming Mode (keeps Discord/OBS)

* FPS Overlay (F1 HUD) - F1 Telemetry-style real-time stats overlay
  - FPS, CPU/GPU temps (F), CPU/GPU usage %, RAM usage
  - Power draw (CPU/GPU/Total wattage)
  - Fan speed indicators
  - Drag to reposition anywhere on screen
  - Chrome bezel racing-inspired design
  - Always-on-top transparent window

* RAM Cache - Redirect temp files to RAM for faster I/O
  - Configurable sizes: 512MB, 1GB, 2GB, 4GB
  - Usage monitoring with progress bar
  - Automatic cleanup on disable

* System Monitoring Widgets (Game Mode page)
  - Fan Speed monitoring - CPU and system fan RPM
  - Power Draw monitoring - CPU, GPU, and total wattage
  - Hardware Sensor Diagnostic Viewer - view all available sensors

IMPROVEMENTS
------------
* Admin Rights Built-in - App runs as administrator automatically
* Fahrenheit Temperatures - All temperature displays now use F
* Tray Exit Fixed - System tray Exit menu properly closes the application
* Updated User Guide - Comprehensive documentation for all features

v2.1.0 - System Tray, Alerts & Historical Graphs
================================================================================

NEW FEATURES
------------

* System Tray Mode
  - Live stats tooltip - hover for CPU/RAM usage
  - Minimize to tray - close button hides to tray (configurable)
  - Quick access menu - right-click for navigation
  - One-click restore - left-click to show window

* Real-time Alerts with toast notifications
  - Temperature alerts - CPU (167F/194F warning/critical), GPU (176F/203F)
  - Memory alert - notification when RAM exceeds 80%
  - Battery alerts - low (20%) and critical (10%) warnings
  - Smart cooldown - 5-minute delay prevents notification spam

* Historical Data & Graphs
  - 30-day history - automatic metric recording every 30 seconds
  - SQLite storage - efficient local database with auto-purge
  - Interactive charts - LiveCharts2 graphs for CPU, Memory, Temperature
  - Time range selector - 1 hour, 24 hours, 7 days, 30 days views

TECHNICAL
---------
* H.NotifyIcon.WinUI for system tray integration
* Entity Framework Core with SQLite for historical data
* LiveCharts2 for interactive charting
