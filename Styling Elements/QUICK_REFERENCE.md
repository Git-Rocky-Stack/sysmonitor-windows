# System Monitor - Quick Reference Card

## Essential Colors (Dark Mode)

| Purpose | Color |
|---------|-------|
| Brand Primary | `#AA2024` |
| Background | `#000000` |
| Card | `#1A1A1A` |
| Text Primary | `#FFFFFF` |
| Text Secondary | `#B3B3B3` |
| Divider | `#333333` |

## Status Colors

| Status | Color |
|--------|-------|
| Success/Good | `#4CAF50` |
| Warning | `#FF9800` |
| Error/Critical | `#F44336` |
| Info | `#2196F3` |

## Corner Radius

| Element | Radius |
|---------|--------|
| Cards | 16px |
| Buttons | 14px (outer), 10px (inner) |
| Inputs | 8px |
| Badges | 4px |
| Progress bars | 4px |

## Typography

- **Font**: Roboto Medium (or Segoe UI Semibold on Windows)
- **Text Primary**: 16px
- **Headings**: 18-32px
- **Labels**: 12-14px
- **Button text**: 14px, uppercase, medium weight

## Button Chrome Effect

```
Layer 1: gradient 135° #E8E8E8 → #A0A0A0 → #606060
Layer 2: gradient 315° #FFFFFF → #C8C8C8 → #888888 (1px inset)
Layer 3: gradient 135° #505050 → #383838 → #282828 (2px inset)
Layer 4: gradient 90° #AA2024 → #C93437 (3px inset)
```
