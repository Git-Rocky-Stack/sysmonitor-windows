Thank you for choosing SysMonitor for Windows!

WHAT THIS SOFTWARE DOES:
------------------------
SysMonitor is a comprehensive system monitoring and
optimization tool that helps you:

  * Monitor CPU, GPU, RAM, and disk usage in real-time
  * Track system temperatures and fan speeds
  * Analyze and optimize system performance
  * Clean temporary files and free up disk space
  * Manage startup programs
  * View detailed hardware information

SYSTEM REQUIREMENTS:
--------------------
  * Windows 10 version 1903 (build 18362) or later
  * Windows 11 (all versions supported)
  * 64-bit processor
  * 100 MB free disk space
  * Internet connection (for updates only)

BEFORE INSTALLATION:
--------------------
  * Close any running instances of SysMonitor
  * Administrator privileges may be required for
    full hardware monitoring capabilities
  * Creating a system restore point is recommended
