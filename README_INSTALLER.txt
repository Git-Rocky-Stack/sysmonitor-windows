ABOUT THIS APPLICATION
----------------------
STX1 System Monitor is a comprehensive Windows system monitoring and
optimization tool that helps you:

  * Monitor CPU, Memory, Disk, Network, and GPU performance in real-time
  * Clean temporary files, browser cache, and system junk
  * Optimize system startup programs
  * Manage running processes
  * Analyze network connections
  * Securely wipe sensitive files
  * And much more!

SYSTEM REQUIREMENTS
-------------------
  * Windows 10 version 1903 (19H1) or later
  * Windows 11 (any version)
  * 64-bit processor (x64)
  * 4 GB RAM minimum (8 GB recommended)
  * 300 MB available disk space

INSTALLATION NOTES
------------------
  * Administrator privileges are required for installation
  * Some features require administrator privileges to function
  * The application is self-contained and does not require .NET Runtime
